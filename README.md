# 🧹 Data Cleaning & Transformation  

## 📌 Project Overview  
This repository contains a dataset of **multi-country product sales** with attributes such as manufacturing date, country code, brand, price, quantity, and category. The project focuses on **data cleaning and transformation** to prepare the dataset for analysis and visualization in tools like **Power BI, Tableau, and Excel**.

## 📊 Dataset Features  
- **Manufacturing Date** – Standardized to `MM/DD/YYYY` format  
- **Country Code** – ISO codes (US, UK, IN, CA, etc.)  
- **Product Brand** – Normalized brand names (Dell, Nike, Samsung, etc.)  
- **Price ($)** – Cleaned numeric values  
- **Quantity** – Validated integers  
- **Category** – Grouped into consistent categories (Electronics, Fashion, Kitchen, Accessories, Outdoor)  

### Sample Records  
```
01/28/2026 | US | Dell Laptop | $1000 | 30 | Electronics  
06/07/2026 | UK | Sony Headphones | $594 | 45 | Electronics  
09/29/2026 | CA | Google Smartphone | $800 | 45 | Electronics  
07/09/2026 | FR | Casio Watch | $100 | 20 | Accessories  
```

## 🛠️ Data Cleaning Steps  
1. Converted inconsistent date formats into a uniform structure.  
2. Standardized country codes to ISO format.  
3. Normalized product categories for consistency.  
4. Validated price and quantity fields.  
5. Checked for duplicates and anomalies.  

## 📈 Transformation Goals  
- Enable **sales trend analysis** across countries and categories.  
- Prepare data for **interactive dashboards**.  
- Support KPIs such as:  
  - Total Sales Amount  
  - Total Quantity Sold  
  - Category-wise Contribution  
  - Country-level Insights  

## 🚀 Usage  
- Import the dataset into **Power BI, Tableau, or Excel**.  
- Apply transformations using **SQL / Python (Pandas)**.  
- Build dashboards to visualize:  
  - 📊 Category-wise sales  
  - 🌍 Country-level product distribution  
  - 💰 Price vs. Quantity trends
